# Assistente de Preenchimento de Formulários

Este app preenche automaticamente 3 formulários `.odt` via interface web (Streamlit).

## Como subir para seu GitHub

1. Acesse https://github.com/Regislainec/assistente-formularios-rdp-rmp
2. Clique em **Add file > Upload files**
3. Envie todos os arquivos descompactados do `.zip` fornecido.
4. Clique em **Commit changes**.

## Como publicar no Streamlit Cloud

1. Acesse https://streamlit.io/cloud e clique em “New app”.
2. Escolha seu repositório `assistente-formularios-rdp-rmp`.
3. Deixe como script principal: `streamlit_app.py`
4. Clique em **Deploy**.

Você receberá um link público e funcional para usar o app no navegador.
